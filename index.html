Project Idea

Overview

My project idea is to create a Train/Bus ticket app for the iPhone, for some reason there is a myki app for android devices however Apple users are hung out to dry. My idea is to create an app suitable for the Apple device and allow people from all over Melbourne to move from pulling a card out of your wallet to simply tapping your phone on the scanner. This project will allow the other 54% of Australians to use Myki on their phones and aim to improve the convenience and environmental impact of using public transport.

Motivation

Statistics from the Global Statistics Counter from February 2021 show that over 54% of Australians are using IOS devices and the other 46% are using android, windows and blackberry devices. From these statistics we simply ask why is there not an app for the IOS? Creating an app like this will also reduce the waste when their Myki cards become unused or thrown away since everything will be moved to an electronic stage. This is a fairly simple but very needed app Australians all need and want which can make everyday life easier for the usual public transport goers.

Description

My project explores an improvement to an already existing application which is only exclusively available on android phone operating systems. My aim is to allow the other 54% of Australians to be able to use the myki on their phones by creating an app which allows the user to simply scan their phone to use public transport. The benefits of using an app over the outdated card is convenience and quickness, people won't have to top up their myki at the station but can do it beforehand at home. Another aim is to try reduce the waste produced and make using public transport more eco-friendly. My app will also allow users to top up, check balance and order new cards (if they still prefer physical cards). All of these features just mentioned are currently all being done face to face and there is no other way to do it, this is a huge inconvenience as people must do it face to face therefore this app will be saving users a lot of time since they can do it at home or even on the way towards the station. Another benefit to my project is that it will reduce the amount of waste that physical cards produce such as cards not in use and paper receipts. These cards are made of plastic and are just another thing that will be thrown into the landfill and paper receipts given from topping up all go into the bin anyways.

The app is fairly simple and will contain many features that will make using public transport a lot easier than it currently is. When opened, the application shows the card on screen and immediately the user will be able to scan their phone onto the terminal and depending if they have money or not will be allowed to pass through the gates. The other features on the app include an easy top up system where the user will input their card / paypal / payment details and input as much money as they want. A limitation of the physical terminal is that it only allows for 1 user at a time, waiting in line for one of these terminals may cause users to miss the train, therefore allowing online transactions that top up the account will remove an issue like this.

In addition, this card will work with the already up and running myki website created by PTV Victoria. So myki cards on ios can be updated on the website just like android, and apple users will be able to create an account which is unique to them. The app stores personal information which is needed to know what card the user is eligible to receive. Being under 18 or a student you may get a concession card, 60+ elderly card, full fare and so on. Payment information may also be stored so users can top up. Storing personal confidential information requires strong cyber-security so I will also be implementing a strong security encryption system that protects all user data.

Tools and Technologies

Creating a website and app requires mainly only 2 hardware tools, a computer and an iPhone. The computer we need must be able to run the latest coding softwares and we need an iPhone to test the application on. Coding softwares needed to create a website that can store and save information include notepad++ and SQL. The coding software for the application can vary however will mainly be looking for a person who can use one of these languages; Swift, Xcode, appcode or coderunner as these are the most popular and refined. Another tool or service we will need are a sample group to test in the real world situation. This group will be the beta testers and will use the app and report problems back to the developers.

Skills Required

To create the app the skills required from an app developer team would be proficiency in any of these main ios coding languages such as Swift, Xcode, Appcode and Coderunner, however to create the website we must also find people who are skilled in html and SQL. User interface and user experience professionals are needed to help make the app / website easy and intuitive to use, these experts will also improve the design and look of our project. Lastly an expert in security is needed to keep peoples details safe as people are putting their card information into the website/app. Finding these skills will be easy as there are many skilled app developers all around the world and we are not limited to Australia.

Outcome

This project will overall improve the whole experience of taking public transport by moving all the physical aspects digital. This is a great innovation to an already outdated system which will not only provide better accessibility but also provide a better eco friendly future. Giving the Myki app on the iPhone / iOS will bring nothing but aid to our community and our country.
